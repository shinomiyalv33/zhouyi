# 周易占卜 · 使用说明

## 如何打开

双击 `index.html`，用任意浏览器（Edge / Chrome 等）打开即可，无需安装、无需联网。

## 文件构成

| 文件 | 说明 |
|---|---|
| `index.html` | 应用主程序（界面与起卦、解卦逻辑） |
| `data-01-16.js` ~ `data-49-64.js` | 六十四卦数据库（卦辞、象传、爻辞及白话解读） |

所有文件须放在同一文件夹内。

## 功能

### 1. 铜钱起卦（六爻法）
仿古法「三枚铜钱掷六次」：每掷一次成一爻，自下而上累成六爻。
- 字（阴面）计 2，背（阳面）计 3；三枚之和为 6（老阴 ✕）、7（少阳）、8（少阴）、9（老阳 ○）
- 老阴、老阳为**动爻**，动则变，由此得**变卦**
- 可逐次点「掷卦」体验过程，也可「一键成卦」

### 2. 时间起卦（梅花易数）
按《梅花易数》之法：年支数 + 月 + 日 之和除以八取上卦，再加时支数除以八取下卦，总数除以六取动爻。传统以农历为准，可自行填入农历数值。

### 3. 数字起卦（梅花易数）
心中默想所问之事，随意报两个数：第一数取上卦、第二数取下卦、两数之和取动爻。

### 4. 寻物占（梅花易数 · 占失物）
丢了东西时使用：默念所失之物，报两数起卦（急寻也可直接以当下时刻起卦），应用依古法自动推断——

- **方位**：动爻所在之卦为「用卦」，以用卦配后天八卦方位（乾西北、坎北、艮东北、震东、巽东南、离南、坤西南、兑西），罗盘图直观指示；
- **场所**：按用卦类象推断失物所处环境（如坎为水边湿处暗处、艮为墙角柜箱、离为明处文书电器旁……）；
- **高低远近**：按动爻爻位断（初爻低处贴地……上爻高阁远方）；
- **可否寻回**：按体用五行生克断——用生体/比和可寻回，体克用费力可得，体生用宜速寻，用克体难寻；
- **再寻之方**：动爻变后用卦所化之卦，指示第二方位。

### 5. 六十四卦一览
完整卦库浏览，点击任意一卦查看卦辞、象传、六爻爻辞与白话解读。

## 断卦规则（依朱子之法，应用会自动提示）

| 动爻数 | 断法 |
|---|---|
| 0 | 以本卦卦辞断 |
| 1 | 以动爻爻辞断 |
| 2 | 以上动之爻为主 |
| 3 | 本卦、变卦卦辞并参 |
| 4~5 | 以变卦为主断 |
| 6 | 乾用「用九」、坤用「用六」，余卦以变卦卦辞断 |

## 所问断语（针对所问之事的结论）

在「所问何事」中写下你的问题（如"最近想跳槽合适吗"），起卦后结果最上方会给出**所问断语**：

1. **识别问事类别** —— 自动归类为事业谋望 / 学业考试 / 感情婚恋 / 财运求财 / 健康平安 / 出行迁移 / 词讼纠纷 / 寻人寻物 / 人际合作；未填则按综合运势断。
2. **选定主断文本** —— 依上表断卦规则，自动选出该以哪段经文为主（卦辞或动爻爻辞），并列明依据。
3. **吉凶定级** —— 对主断经文中的吉凶用语（元亨、大吉、吉、亨、利、无咎、悔、吝、厉、凶……）加权评分，定为：上吉 / 吉 / 中平 / 谨慎 / 不宜 五级，给出明确结论。
4. **卦势走向** —— 比较本卦与变卦的吉凶趋势（先难后易 / 先易后难 / 平稳相承），并按动爻爻位（初爻事之始……上爻事之极）提示进退分寸。
5. **就此事之建议** —— 引用与问事类别对应的解读，并附变卦所示的后续走向。
